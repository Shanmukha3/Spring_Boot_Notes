1. Bootstrap5 features:
	1. uses Grid System
	2. Most popular CSS framework
	3. Well documented
2. Bootstrap4 VS Bootstrap 5:
	1. More colors are added to Bootstrap 5
	2. floating labels(like place holder) in Bootstrap 5
	3. Doesn't support the internet Explorer browser
	4. it supports RTL(Right to Left) typography also
	5. can load placeholder easily
	6. Customization 
	7. SVG(means notifications symbols) icons
3. Container Classes:
	1. .container : fixed responsive
	2. .container-fluid : fluid responsive(entire width of viewport)
	<div class="container/container-fluid">msg</div>
4. Grid system:
	1. each row will be having 12 columns.
	2. <div class="row"><div>
	3. <div class="col-lg-6"></div>
	4. <div class="col-lg offset-2"></div>

	1. lg  (for large devices with screen width >= 1200px)
	2. md (for medium devices with screen width < 1200px and >=992px)
	3. sm (for small devices with screen width < 992px and >=768px)
	4. xs (for extra small devices with screen width < 768px)
	
	1. overflow: when no of grids exceed 12 then that column will be sent to new column
	2. offset: if we want to give some space between two columns then we can use this

	1. If multiple images to be fit based on screen:
		 <div class="row">
    			<div class="col-<sq1>-<n> col-<sq2>-<m>"> â€¦ </div>
  		</div>

		4. .text-warning	-yellow

6. Table:
	1. .table	- horizontal dividers and small cell padding ( 8px ). 
	2. .table table-bordered   - to give border
	3. .table table-sm       - To make the table more compact
	4. .table table-hover    -  enables the hover state of the table rows. 
	5. .table-responsive	 - For making a table responsive in nature we should put the table inside a <div> tag which has the Bootstrap class .table-responsive.
		<div class="table-responsive">
	6. <table class="table table-striped">   - to provide alternative coloring like zebra crossing
7. Images:
	1. class="rounded"	:  add rounded corners to the image
	2. class="rounded-circle": give the image a circular shape
	3. class="img-thumbnail" : give thumbnail appearance to an image
	4. class="img-fluid"   :  can automatically adjust its size with respect to the screen size.
		In bootstrap it is having as  max-width: 100%; height: auto;

	2. styling radio and checkbox:
