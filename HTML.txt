Protocol:
1. there should be some rules btw serverside and web browser side to communicate safiely such rules are called as protocol. one among them is HTTP

HTTP:
1. hypertext transfer protocol
2. HTTP is a request/response based stateless(each request is executed independently) and connectionless(because for every request new connection is established) protocol
3. it establies request based on user to server side, response is sen din same connection, once it is delivered then connection will be terminated

HTTPS:
1. the data that was exchanged in http connection can be listen easily by gaining access to the network. so to overcome this HTTPS came.
2. Hyper text transfer protocol over secured socket layer. communication is encrypted.

6. Attributes of forms:
	1. required-	Value required for form submission.
	2. value-	Specifies default value of input element.
	3. type-	Identifies type of input element.
	4. name- 	Name of input element used for form submission.
	5. size-	Identifies the width of input text that the user can see.
	6. autofocus-	Focuses on a particular form element automatically.
	7. maxlength-	Identifies the maximum length of input value.
	8. minlength-	Identifies the minimum length of input value.
	9. pattern-	Specifies a pattern (regular expression) for entering input text. 
	10.placeholder- Displays a text (hint) within an input control for the user.
	11. steps-	Specifies the legal number of intervals.
	12. formnovalidate- Skips validations for forms upon form submission, used with submit button. 
	novalidate-	skip validation for fields in form
	13. readonly-	Prevents user from modifying the value, value will passed on submit
	14. disabled-	Prevents from modifying the value, value is not sent on submit
	15. multiple-	 allows to enter/select more than one value

Anchor:

1. <a href="url" target="value">Text</a>
	1. An unvisited link is underlined and blue
	2. A visited link is underlined and purple
	3. An active link is underlined and red
2. attributes of anchor tag:
	 we want keep in the target
	1. _blank : open in new window
	2. _self  : open in same window
	3. _parent: open in the parent of the current window
	4. _top   : open in the top most level window of the current window
3. when using _blank then there is problem of security so to overcome that we will keep link rel="noreferrer noopender"
4. to get info about anchor tag we will have title attribute for anchor tag.
    <a href="link" title="searching something in google"> Google </a>
